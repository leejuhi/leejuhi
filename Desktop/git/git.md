## git 정리 하기

# 터미널 명령어

- mkdir 파일이름 (파일 생성)
- cd 파일이름 (파일 안으로 이동)
- touch 파일이름 -빈 파일 생성

# 사용자 설정

- 이름: git config user.name “”
- Email: git config user.email “”

# Commit file 반영

- git add 파일이름 or 전체 업뎃시 .->git commit -m “메시지“
- git status git 상태보기
- git reset 파일 이름 - staging area에서 삭제
- git push: local 에서 바뀐 내용을 remote 해줌
